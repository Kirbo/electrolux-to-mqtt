# Security Policy

## Supported Versions

Only the latest release receives security fixes.

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
| Older   | No        |

## Reporting a Vulnerability

Please do **not** open a public issue for security vulnerabilities.

Email: kirbownz@me.com

Include a description of the issue, steps to reproduce, and the potential impact. Please allow time to patch before any public disclosure.

**Timeline:**
- Acknowledgement: within 7 days
- Fix: within 30 days of acknowledgement
